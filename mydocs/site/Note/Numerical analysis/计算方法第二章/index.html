    计算方法第二章¶

    本科阶段我修读了计算方法这门课程，在完成习题的过程中发现绝大部分的题目都需要借助计算机程序进行求解，正好目前在学习CS61A（刚刚刷完递归），借此来锻炼我在数值计算方面的编程能力。

    Newton迭代法¶

    题目是这样的：用牛顿法求解下列非线性方程组的最小正根，初值\(x_0\) = 2.5 $$ cosx + \frac{1}{1+e^{-2x}}=0 $$ 牛顿迭代法需要求导操作，我不想直接把导数填在程序中，希望求导操作能让程序自己完成，查询后我了解到需要导入sympy库，其中diff可以对输入的函数式进行求导操作。

>>> df

    def Newton(x):
    def f(x):
        return cos(x) + 1 / (1 + exp(-2))

    def df(x):
        return diff(f(x), x)

    update_x = x - f(x) / df(x)
    difference = abs(update_x - x)
    eps = 0.0000005
    if difference > eps:
        return Newton(update_x)
    else:
        return update_x

    问题出在if difference > eps:，报错信息为：

    Text Only
    1
    TypeError: cannot determine truth value of Relational

    后来发现difference并不是一个确切的值，而是一段符号（因为通过diff求导得到的不是一个函数，而是象征着导数形式的一串符号），查找资料后发现如果想对diff求导后的式子赋值（也就是说让它变成一个能接受传参的函数），需要用到.subs

    Python
    1
2
3
    >>> df = diff(cos(x), x).subs(x, 0)
>>> print(df)
0

    现在传参问题解决了，并且能成功得到答案，但目前这个程序只能求解此题，也就是说如果我想求解另一个方程，我需要打开这个.py文件修改其中的方程，初始值，以及判断条件才能运行。于是我想能不能优化这个程序，实现只需要用户在命令行输入方程，初始值，判断值（就像一个计算器）就能输出相应内容，开始实践

    我先写好了函数名，注意这里的function应该是一段符号（输入的方程）（这里需要在函数调用之前就要将x变成符号），需要我们稍后将其转化为能接收参数的函数

    现在需要在这个函数下定义原函数与导函数：

    Python

        def f(x):
        x_sym = symbols('x')
        return function.subs(x_sym, x)

    def df(x):
        x_sym = symbols('x')
        return diff(f(x_sym), x_sym).subs(x_sym, x)

    x_sym = symbols('x') 的目的是创建一个名为 x_sym 的符号对象，这样function中的x在接下来会被视为x_sym，然后function.subs(x_sym, x)会将x的值（传入到f中的）传递给x_sym，也就实现了将x的值带入到function中的目的

    from sympy import *

def Newton(function, x, eps):
    def f(x):
        x_sym = symbols('x')
        return function.subs(x_sym, x)

    def df(x):
        x_sym = symbols('x')
        return diff(f(x_sym), x_sym).subs(x_sym, x)

    update_x = x - f(x) / df(x)
    difference = abs(update_x - x)
    print(update_x)

    return Newton(function, update_x, eps) if difference > eps else update_x

x = symbols('x')

    from sympy import *

def aitken(x, y):
    x_1 = acos(-(1 / (1+exp(-2*x))))
    x_2 = acos(-(1 / (1+exp(-2*x_1))))
    update_y = x - ((x_1 - x) ** 2) / (x_2 - 2*x_1 + x)
    print(update_y)
    return aitken(x_1, update_y) if abs(update_y - y) > 0.000005 else 0

    from sympy import *

def steffensen(x):
    def f(x):
        return acos(-(1/(1 + exp(-2*x))))

    s = f(x)
    t = f(s)
    update_x = x - (s - x)**2 / (t - 2 * s + x)
    print(update_x)
    return steffensen(update_x) if abs(update_x - x) > 0.000005 else update_x

    from sympy import *

def func(lower, upper):
    def f(x):
        return 35 * tan(x) - (12017.25 / 784.08) * (1 / (cos(x)**2)) + 1

    update_x = round((lower + upper) / 2 , 5)
    if abs(update_x - upper) > (0.5)**10:
        if f(update_x) * f(upper) < 0:
            print('[%f, %f], %f' %(update_x, upper, (upper + update_x)/2))
            return func(update_x, upper)
        else:
            print('[%f, %f], %f' %(lower, update_x, (lower + update_x)/2))
            return func(lower, update_x)
    else:
        return update_x

    要注意的是格式化输出与round函数的使用
